## Installation - Running locally

Pre-requisits:
You need npm package manager.

1. Clone the repo;
2. Start the server:
  ```
  cd server
  npm i
  npm run dev
  ``` 
3. Start the web:
  ```
  cd web
  npm i
  npm run dev
  ``` 
  
3. Start the mobile:
  ```
  cd mobile
  npm i
  npx expo start
  ```
